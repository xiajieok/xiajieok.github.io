Mac redis

brew install redis
开机启动
brew services start redis

配置文件
/usr/local/etc/redis.conf

编辑配置文件 配置后台运行
daemonize yes

服务端启动及查看

客户端调用
redis-cli

给redis设置密码
打开redis.conf文件，找到以下配置项
# requirepass foobared
更改为
requirepass 你的密码
注意：由于上诉操作更改了redis.conf文件，所以下次再启动的时候，要手动加载一下redis.conf文件。例如
进入到redis根目录，启动如下
redis-server /usr/local/etc/redis.conf
